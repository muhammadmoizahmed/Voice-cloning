Access Levels & Permissions

Administrator (Full Access)

  • ✓ Create & manage sub-admins/moderators
  • ✓ Delete any user including admins
  • ✓ View all analytics & financial reports
  • ✓ Manage user roles (promote/demote)
  • ✓ Voice moderation & verification
  • ✓ Change own password
  • ✓ Full system configuration access

Moderator (Limited Access)

  • ✗ Cannot create sub-admins
  • ✗ Cannot delete admin users
  • ✓ View analytics & reports
  • ✓ Suspend/activate regular users
  • ✓ Voice moderation & verification
  • ✓ Change own password
  • ✓ Manage user-generated content
