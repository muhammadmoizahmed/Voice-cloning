## рҹҺҜ Component Responsibilities

### Frontend Components
- **Authentication**: Login, registration, OTP verification
- **Dashboard**: User stats, quick actions, navigation
- **Voice Management**: Upload, clone, manage voices
- **Audio Generation**: Text-to-speech interface
- **Avatar Video**: Face detection + video generation
- **Admin Panel**: User management, analytics, settings

### Backend Components
- **Auth Router**: User authentication, JWT tokens
- **Voice Router**: Voice upload, cloning, management
- **Generation Router**: Audio generation, downloads
- **Payment Router**: Stripe integration, webhooks
- **Admin Router**: User management, analytics
- **Face Detection**: Image processing, validation

### External Services
- **ElevenLabs**: Voice cloning and TTS
- **Fish.Audio**: Alternative TTS service
- **Resemble AI**: Voice cloning service
- **Stripe**: Payment processing
- **SMTP**: Email notifications
- **Redis**: Session caching (future)
